Assignment
---------------------------

INSTALLATION:
-------------
  1. Install as you would normally install a Drupal module. 

REQUIREMENTS
------------
  Nothing special is required.

CONFIGURATION
-------------
  1. Install the module "Assignment".
  2. Admin config Url:- admin/config/assignment/site-settings
  3. Fill value for Country, City and Timezone field and submit the form.
  4. Go to the “Block Layout”. Eg:- Admin Menu >> structure >> block layout
  5. Go to your block region where you want to put it.
  6. Click the "Place block" button and in the modal dialog click the 
     "Place block" button next to "Site Config Block".
  5. Click on the Save block button.

UNINSTALLATION
--------------
  1. Uninstall as you would normally Uninstall a Drupal module.
